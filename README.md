# Python Roadmap

A learning repository for practicing Python fundamentals through small projects, notes, and exercises. The current work focuses on core syntax, functions, file storage, JSON, modules, and simple command-line programs.

## Overview

This repository documents my Python learning path week by week. It is not presented as a production application; it is a transparent record of practice, iteration, and growth.

## Current Project

Week 1 contains a small **student management CLI** that stores student records in JSON and supports adding, listing, searching, updating, and removing students.

```text
python-roadmap/
├── data/
│   └── students.json
└── week1/
    ├── main.py
    ├── models.py
    ├── storage.py
    ├── student_manager.py
    ├── notes.txt
    └── README.md
```

## Skills Practiced

- Python functions and modules
- Basic input validation
- Working with dictionaries and lists
- Reading and writing JSON files
- Separating storage logic from application logic
- Building a command-line menu
- Documenting concepts while learning

## Run Week 1

```bash
git clone https://github.com/haseebahmad60/python-roadmap.git
cd python-roadmap
python week1/main.py
```

The program opens an interactive menu in the terminal.
